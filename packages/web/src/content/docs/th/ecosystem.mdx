ชุดโครงการชุมชนที่สร้างขึ้นบน Cody

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Cody ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-cody](https://github.com/awesome-cody/awesome-cody) และ [cody.cafe](https://cody.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
